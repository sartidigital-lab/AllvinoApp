# STACK.md

**Date:** 2026-05-17

## Overview
This project is a static frontend web application structured as a Progressive Web App (PWA).

## Technologies

- **HTML5:** Core structure of the application.
- **CSS:** Styled entirely using **Tailwind CSS** via CDN (`https://cdn.tailwindcss.com?plugins=forms`).
- **JavaScript:** Vanilla JavaScript (minimal, mostly static HTML pages).
- **PWA:** Basic PWA setup with `manifest.json` and icons.

## Dependencies

No `package.json` is present. The project relies on CDNs.

## Rationale
The stack was likely chosen for rapid prototyping and simplicity, avoiding the overhead of a build step or modern JavaScript framework.
