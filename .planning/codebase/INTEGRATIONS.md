# INTEGRATIONS.md

**Date:** 2026-05-17

## Overview
External services and APIs used by the project.

## Integrations

- **Tailwind CSS CDN:** Used for styling.
- **Google Fonts:** Uses `Noto Serif` and `Manrope`.
- **Google Material Symbols:** Used for icons throughout the application.
- **Unsplash:** Used for placeholder images (e.g., in `index.html`).

## Authentication & Backend
Currently, there is no explicit backend or authentication provider integrated directly into the visible HTML files. Authentication flows (e.g., `login.html`, `admin-login.html`) are likely either mocked or handled via basic static forms right now.
