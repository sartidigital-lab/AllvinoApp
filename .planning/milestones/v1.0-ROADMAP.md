# Milestone Archive: v1.0

**Archived Date:** 2026-05-17
**Status:** Shipped

## Overview
O projeto migrará a aplicação estática (Multi-Page) de E-commerce de Vinhos "Allvino" para uma Single Page Application moderna e performática com Next.js, configurada para ser offline-first (PWA), usando Supabase para autenticação e banco de dados, e Vercel para deploy contínuo via GitHub Actions.

## Accomplishments
1. **Arquitetura Base e PWA:** Estabelecemos uma fundação escalável usando Next.js com Tailwind CSS v4 e PWA offline-first nativo.
2. **Autenticação Segura:** Integramos o Supabase Auth para gerenciar sessões, login, registro e rotas protegidas em todo o aplicativo.
3. **Catálogo Resiliente:** Criamos um catálogo dinâmico de vinhos que busca dados do PostgreSQL, operando mesmo sem internet graças ao IndexedDB.
4. **Fluxo de Compras (E-commerce):** Implementamos um carrinho flutuante, página de checkout com cálculos de frete/retirada, e histórico de compras.
5. **Painel de Administração:** Construímos uma área restrita (`/admin`) com métricas de BI e tabela para gestão e edição (CRUD) do catálogo de produtos e imagens.
