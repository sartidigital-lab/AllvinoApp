# Plan 02-01: Integração Supabase Auth e Componentes de Login/Signup

## Context
This plan implements the `@supabase/ssr` utilities and the UI components for the Auth Modal, handling Email/Password login via Server Actions and redirecting to OAuth for Google. 

## Requirements
- AUTH-01: Login with Email/Password and OAuth Google.
- AUTH-02: UX via Auth Modal available globally.
