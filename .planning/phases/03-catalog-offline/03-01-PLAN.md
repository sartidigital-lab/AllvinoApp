# Plan 03-01: Integração Banco de Dados e Infraestrutura Offline

## Context & Objective
Set up the Supabase database access layer for the wines catalog and install/configure the IndexedDB wrapper (`idb`) to support the offline-first strategy. This establishes the data models and access functions before building the UI.
