# Plan 01-02: Setup PWA

## Goal
Configure the application as a Progressive Web App using `next-pwa`.

## Requirements
- ARCH-03

## Tasks

```xml
<task id="01-02-01">
  <title>Install and Configure next-pwa</title>
  <read_first>
    - next.config.js
  </read_first>
  <action>
    Install dependency: `npm install next-pwa`
    Update `next.config.js` to wrap the Next config with `withPWA({ dest: 'public', disable: process.env.NODE_ENV === 'development' })`.
  </action>
  <acceptance_criteria>
    - `package.json` contains `next-pwa`.
    - `next.config.js` uses `withPWA`.
  </acceptance_criteria>
</task>

<task id="01-02-02">
  <title>Migrate Manifest and Assets</title>
  <read_first>
    - manifest.json
    - src/app/layout.tsx
  </read_first>
  <action>
    Move the existing `manifest.json` from the root to the `public/` directory (create `public/` if it doesn't exist).
    Ensure `src/app/layout.tsx` includes `<link rel="manifest" href="/manifest.json" />` via Next.js Metadata API.
    Move any icons referenced in `manifest.json` into `public/`.
  </action>
  <acceptance_criteria>
    - `public/manifest.json` exists.
    - `src/app/layout.tsx` metadata exports the manifest link.
  </acceptance_criteria>
</task>
```
