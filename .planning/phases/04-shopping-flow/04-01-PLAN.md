# Plan 04-01: Componentes e Estado do Carrinho

## Context & Objective
Implement a global shopping cart state that persists across page reloads using React Context and `localStorage`. Build the Cart Overlay component that slides in from the side (as seen in `catalogo.html`), allowing users to view, adjust, and remove items.
